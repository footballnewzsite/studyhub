🔸 Δομή Υλικού

  • Networking — Πλήρης θεωρία TCP/IP, OSI, LAN/WAN, IPv4/IPv6, routing, firewalls.
  • Wireshark — Ανάλυση πακέτων, φίλτρα, πρακτικά παραδείγματα και ασκήσεις.
  • Standards (ISO / NIST / KΔΠ) — Ανάλυση προτύπων, κανονισμών και compliance.
  • CSIRT / Incident Response — Διαχείριση περιστατικών, forensics, playbooks, workflows.
  • Mock Exams — Ερωτήσεις με απαντήσεις για εξάσκηση.
  • Glossary — Λεξιλόγιο όρων κυβερνοασφάλειας & δικτύων.

🧭 Πώς να μελετήσεις αποτελεσματικά

  • Διάβαζε θεωρία → μετά κάνε πρακτικά παραδείγματα (Wireshark, Volatility, Autopsy).
  • Επανάλαβε τα βασικά πρωτόκολλα και πρότυπα (TCP/IP, ISO 27001, NIST 800-53).
  • Χρησιμοποίησε το Glossary καθημερινά — οι όροι εμφανίζονται συχνά στις εξετάσεις.
  • Κάνε Mock Exams την τελευταία εβδομάδα και αναθεώρησε τις απαντήσεις.

💡 Extra Tips

  • Μάθε τις port numbers (HTTP 80, HTTPS 443, DNS 53, SSH 22).
  • Θυμήσου τα OSI Layers με mnemonic: “Please Do Not Throw Sausage Pizza Away”.
  • Κατανόησε τις διαφορές TCP vs UDP και τα πρωτόκολλα κάθε επιπέδου.
  • Προετοιμάσου για σενάρια Incident Response (π.χ. Phishing, Malware Infection).
  • Εφάρμοσε την PDCA λογική στη διαχείριση ISMS.
